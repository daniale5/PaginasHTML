Fonte Offline

Existem situações que nem sempre podemos incorporar ou importar as fontes diretamente dos endereços do Google Fonts. Aplicativos híbridos, por exemplo, não podem depender de uma conexão com a internet para incoporar as fontes, o mesmo ocorre com aplicações PHP desenvolvidas para intranets. Neste caso, a solução mais interessante seria baixar a fonte do Google Fonts para usá-la offline ao invés de incoporar/importar e criamos uma regra CSS para utilizá-la.
